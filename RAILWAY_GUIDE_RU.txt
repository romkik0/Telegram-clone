┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ШАГ 1: ПРОВЕРКА ЛОКАЛЬНО (5 минут)                            ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. Откройте CMD (Win + R → cmd)

2. Перейдите в папку проекта:
   cd D:\хм

3. Установите зависимости:
   npm install

4. Запустите сервер:
   npm start

5. Откройте браузер и зайдите на:
   http://localhost:3000

6. ✅ Если видите мессенджер - всё работает!

7. Остановите сервер: Ctrl + C


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ШАГ 2: ЗАГРУЗКА НА GITHUB (5 минут)                           ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. В CMD (в папке проекта) выполните:

   git init
   git add .
   git commit -m "Initial commit: Telegram clone"

2. Зайдите на https://github.com

3. Войдите в аккаунт (или создайте новый)

4. Нажмите зеленую кнопку "New" (справа вверху)

5. Repository name: telegram-clone

6. Оставьте все галочки ПУСТЫМИ

7. Нажмите "Create repository"

8. GitHub покажет команды. Скопируйте и вставьте в CMD:
   (ЗАМЕНИТЕ ВАШ_USERNAME на ваш GitHub username!)

   git remote add origin https://github.com/ВАШ_USERNAME/telegram-clone.git
   git branch -M main
   git push -u origin main

9. ✅ Код загружен на GitHub!


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ШАГ 3: ДЕПЛОЙ НА RAILWAY (5 минут)                            ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. Откройте https://railway.app в браузере

2. Нажмите "Login" (справа вверху)

3. Выберите "Login with GitHub"

4. Разрешите Railway доступ к GitHub

5. Нажмите "New Project" (большая кнопка)

6. Выберите "Deploy from GitHub repo"

7. Найдите "telegram-clone" в списке репозиториев

8. Нажмите на него

9. Railway начнет деплой (подождите 1-2 минуты)
   Вы увидите логи деплоя

10. Когда деплой завершится (статус "Success"), нажмите на проект

11. Нажмите "Settings" (слева в меню)

12. Прокрутите вниз до раздела "Networking"

13. Нажмите "Generate Domain"

14. Railway создаст URL типа:
    https://telegram-clone-production.up.railway.app

15. ✅ Скопируйте этот URL!


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ШАГ 4: ЗАЙТИ В МЕССЕНДЖЕР                                      ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. Откройте ваш Railway URL в браузере
   Например: https://telegram-clone-production.up.railway.app

2. 🎉 ВЫ В МЕССЕНДЖЕРЕ!

3. Что можно делать:

   ✅ Нажмите на любой чат слева
   ✅ Напишите сообщение внизу
   ✅ Нажмите кнопку отправки (➤)
   ✅ Нажмите ☰ (меню) для открытия профиля
   ✅ Измените имя, загрузите фото
   ✅ Включите темную тему в настройках
   ✅ Создайте новый канал

4. Поделитесь ссылкой с друзьями!

❌ "npm не является внутренней командой"
   → Установите Node.js с https://nodejs.org

❌ "git не является внутренней командой"
   → Установите Git с https://git-scm.com

❌ Railway просит карту
   → Это нормально, карта для верификации
   → Railway дает $5 бесплатных кредитов в месяц
   → Списаний не будет если не превысите лимит

❌ Белый экран на Railway
   → Подождите 1-2 минуты (первый запуск медленный)
   → Проверьте логи в Railway (Deployments → View Logs)

❌ git push просит пароль
   → GitHub больше не принимает пароли
   → Используйте Personal Access Token:
     GitHub → Settings → Developer settings → 
     Personal access tokens → Generate new token
